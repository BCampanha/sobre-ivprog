About iVProg

The iVProgH is a free educational system, implemented in HTML5 (strongly uses JavaScript), is intended for teaching and learning algorithms and programming. Its web version has a scheme for automatic evaluation of exercises. IVProg was designed from a visual model for programming, and can be used since the second cycle of elementary school.
