Commands

Commands are used to create the logical structure of a program. In iVProg you can add the following commands:

  • Main Commands
    • data input (read);
      •       
    • data output (print);
      •       
    • comment insertion;
      •       
    • assignment (set value for some variable);
      •       
    • call function (used to generate a value);
      •       
    • selection command;
      •       
    • fixed loop command (type loop for);
      •       
    • repeat command with input condition (loop of type repeat while);
      •       
    • repeat command with output condition (type loop repeat until);
      •       
    • selection command from several options;
      •       
    • return command (used in functions).

  • Contextual Commands
    All repetition command allow a special command to "break" the repetition process, that means, if it is executed, the loop is immediately interrupted (Stop).
    Another special command is the case, that must be used inside the switch command, as presented in figure 2.
    • break
    • case
