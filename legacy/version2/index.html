iVProg : Visual Programming (LInE-IME-USP)

LInE. Software educacional livre. Interatividade na Internet para aprendizagem.

O iVProg é um sistema para ensino-aprendizagem de algoritmos e programação, que pode ser usado desde o ensino fundamental II até o ensino superior. Para isso, ele implementa o paradigma de Programação Visual e permitir a avaliação automática de exercícios (a partir de casos-de-teste).
Nós também disponibilizamos uma versão em HTML5, que pode ser examinada a partir desse endereço).
Ambos os sistema são livres e seus códigos fonte podem ser examinados a partir dos seguintes endereços: iVProg2 no GitHub e iVProgH no GitHub.

Se tiver problemas para testar esta versão do iVProg dentro de seu navegador, pode-se descarregar o pacote iVProg e rodá-lo como aplicativo Java em seu computador. Para isso clique aqui e grave-o em seu computador. Se tiver o Java instalado, basta um "clique" duplo sobre o mesmo para iniciá-lo.

Para adicionar variáveis clique no botão . Para comandos no botão . Para interpretar seu código, clique no botão (no canto inferior esquerdo, o botão do meio).
Para examinar um código pronto clique aqui.
