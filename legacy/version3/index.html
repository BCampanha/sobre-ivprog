iVProg : Visual Programming (LInE-IME-USP)

LInE. Software educacional livre. Interatividade na Internet para aprendizagem.
O iVProgH é mais um sistema educacional livre do LInE, implementado em HTML5 (usa fortemente JavaScript), este destinado ao ensino e aprendizagem de algoritmos e programação. O iVProgH é uma nova implementação do iVProg que é implementado em Java. Ambos iVProg dispõem de um esquema para avaliação automática de exercícios, experimente!
Esses sistemas foram desenhados a partir de um modelo visual para programação, podendo ser utilizado desde o segundo ciclo do Ensino Fundamental.
Para adicionar variáveis "clique" no botão + Variáveis. Para inserir um comando, "clique" no botão + Comando escolhendo-o. Para interpretar seu código, "clique" no botão (abaixo, ao lado do botão "Limpar console").
Ver alguns outros exemplos de codigo no iVProgH: programa para imprimir n primeiros numeros de Fibonaccii ; exercicio 2: ler 2 valores e imprimir sua soma ; Ver solucao do exercício 2.
Sobre funcionamento do iVProgH
Exercício 1: Fazer um algoritmo que solicite que o usuário digite 2 valores (inteiros), depois compute a soma dos valores digitados e por último imprima esta soma.
Algumas explicações adicionais:
  • Na versão do iVProgH apresentada abaixo, aparece na parte inferior o gabarito utilizado para realizar a avaliação automática. É possível suprimir isso eliminando o rótulo &iLM_PARAM_SendAnswer=true dos parâmetros (examine o código HTML desta página para entender isso).
  • Após finalizar seu algoritmo "clicar" no botao "Enviar resposta".
