iVProg - Interactive Visual Programming in the Internet

The iVProg - Interactive Visual Programming in the Internet is a new system targeting the teaching-learning of algorithms through the World Wide Web (Web). Thes iVProg is the Alice system, a free software from Carnigie Mellon University
The iVProg has been developed since 2009 June. You can check its prototype in the applet bellow.

Example nº 1

A program such that, given a natural number K (provide by user), print the information if it is prime or not.

Example nº 2

A program such that, given a natural number K (provide by user), print the sum of the first K naturals.
